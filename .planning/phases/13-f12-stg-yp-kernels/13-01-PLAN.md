    a. `COS_14_14` static table: Copy the 196-element `static const double COS_14_14[]` array exactly from `stg_roots.c` lines 17-212.

    b. `_clenshaw_dc(im: &mut [f64], coeff: &[f64], x: f64, n: usize)` -- Clenshaw recurrence over the u-axis. Port from `stg_roots.c` `_clenshaw_dc` function. For each of the `n` output values, performs Clenshaw evaluation of degree-13 Chebyshev polynomial using the 14 coefficients at stride `n`.

    c. `_matmul_14_14(out: &mut [f64], inp: &[f64], n: usize)` -- Matrix-vector multiply with `COS_14_14`. Port from `stg_roots.c` `_matmul_14_14`.

    d. `_clenshaw_d1(out: &mut [f64], coeff: &[f64], x: f64, n: usize)` -- 1D Clenshaw evaluation over the t-axis. Port from `stg_roots.c` `_clenshaw_d1`.

    e. `pub fn stg_roots_host(nroots: usize, ta: f64, ua: f64) -> (Vec<f64>, Vec<f64>)` -- Main entry point. Algorithm per D-05/D-06/D-07/D-08:
       - `let t = ta.min(19682.99_f64);` -- EXACT clamp per D-07, `const T_MAX: f64 = 19682.99_f64;`
       - if `t > 1.0`: `tt = t.ln() * 0.9102392266268373 + 1.0`; else: `tt = t.sqrt()`
       - `uu = ua.log10()`
       - `it = tt.floor() as usize; tt = 2.0 * (tt - it as f64) - 1.0`
       - `iu = (uu + 7.0).floor() as usize; uu = 2.0 * (uu - (iu as f64 - 7.0)) - 1.0`
       - Offset: `let off = nroots * 196 * (iu + it * 10);`
       - Call `_clenshaw_dc`, `_matmul_14_14`, `_clenshaw_d1` for roots using `DATA_X[off..]`
       - Same pipeline for weights using `DATA_W[off..]`
       - Final: `weights[i] /= ua.sqrt()` for each weight
       - Return `(roots, weights)`
