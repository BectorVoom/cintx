Replace the stub in center_3c2e.rs with a real 3c2e kernel (per D-01, D-03, D-12, D-13).

**Key insight (Pitfall 4):** The 3c2e integral has 3 shells (i, j, k) but reuses the 2e G-tensor machinery by mapping:
- "ij" side = actual shells i, j (bra pair)
- "kl" side = actual shell k mapped to the "l" slot with the "k" slot being an s-function (lk=0, ak=0 effectively)
- From g3c2e.c: `ll_ceil = k_l` (the third real shell's angular momentum goes into the ll slot)
- `lk_ceil = 0` in the 2e sense (the "k" slot is an s-function phantom)
- `akl = ak` (the third shell's exponent)

**Rys root count:** `nrys = (li + lj + lk) / 2 + 1` (from g3c2e.c).

**G-tensor layout (from CINTinit_int3c2e_EnvVars):**
```
ibase = (li_ceil > lj_ceil)  // same 2e adaptive stride
if ibase:
  dli = li_ceil + 1
  dlj = lj_ceil + 1
else:
  dli = lj_ceil + 1
  dlj = li_ceil + 1
dlk = lk_ceil + 1  // = k_l + 1 (the actual third shell angular momentum)
g_stride_i = nrys
g_stride_k = nrys * dli  (this is g_stride_l in 2e terms, but it's actually the third-center k)
g_stride_j = nrys * dli * dlk
g_size = nrys * dli * dlk * dlj
```

**Primitive loops:** Triple loop over ip=0..nprim_i, jp=0..nprim_j, kp=0..nprim_k:
1. Form ij pair: `aij = ai + aj`, `P = (ai*A + aj*B) / aij`
2. Third center: `akl = ak` (single primitive, no l shell), `Q = C` (shell k center)
3. Compute `rho = aij * akl / (aij + akl)`, `x_rys = rho * |P - Q|^2`
4. Get Rys roots/weights: `rys_roots_host(nrys, x_rys)`
5. For each Rys root t^2 = u[r]:
   - Compute 2e-type recurrence coefficients:
     - `b10[r] = 0.5 * (1 - akl * u[r] / (aij + akl)) / aij`
     - `b01[r] = 0.5 * (1 - aij * u[r] / (aij + akl)) / akl`
     - `b00[r] = 0.5 * u[r] / (aij + akl)`
     - `c00[r][axis] = P[axis] - A[axis] + b00[r] * (Q[axis] - P[axis])` (shift to i center if ibase)
     - `c0p[r][axis] = Q[axis] - C[axis] + (akl/(aij+akl)) * u[r] * (P[axis] - Q[axis])`
       But Q = C for 3c2e, so `c0p[r][axis] = (akl/(aij+akl)) * u[r] * (P[axis] - Q[axis])`
   - Fill gz base: `gz[root] = w[r] * PI^(3/2) / (aij * akl * sqrt(aij + akl)) * exp(-ai*aj/aij * |A-B|^2) * exp(-rho * |P-Q|^2)`
     The `exp(-ai*aj/aij * |A-B|^2)` comes from `compute_pdata_host(ai, aj, A, B).fac`.
   - VRR upward in i-direction (or j if !ibase) using b10, c00
   - VRR upward in k-direction (mapped to "l" slot) using b01, c0p
   - Cross-recurrence b00 connects i and k dimensions
   - If needed, HRR to move from the ibase dimension to the actual i/j split

6. **Contract** Cartesian components over `cart_comps(li) x cart_comps(lj) x cart_comps(lk)`.

7. Apply `common_fac_sp(li) * common_fac_sp(lj) * common_fac_sp(lk)` scaling.

8. If Spheric: call `cart_to_sph_3c2e(cart_buf, li, lj, lk)`.

9. Write to staging (i fastest, k slowest ordering), return ExecutionStats.

**Output ordering:** i fastest, then j, then k slowest (column-major matching libcint 3-index ordering).
