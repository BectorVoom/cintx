All three gaps from the initial verification are closed:

**Gap 1 (Blocker - CLOSED): Oracle tests now compare against vendored libcint 6.1.3.**
Plan 09-04 compiled libcint 6.1.3 from 16 C source files via the cc crate in `cintx-oracle/build.rs`, generated FFI bindings via bindgen, and added three vendor parity test functions (`test_int1e_ovlp_sph_h2o_sto3g_vendor_parity`, `test_int1e_kin_sph_h2o_sto3g_vendor_parity`, `test_int1e_nuc_sph_h2o_sto3g_vendor_parity`). Two algorithm bugs were found and fixed during this work: the kinetic energy derivative used D_i^2 (bra direction) instead of D_j^2 (ket direction), and C2S_L1 used py/pz/px ordering instead of the identity px/py/pz. With these fixes all three vendor parity tests pass with mismatch_count == 0.
