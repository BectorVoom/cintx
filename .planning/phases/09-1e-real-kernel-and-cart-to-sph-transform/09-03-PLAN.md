    - `bas[ATOM_OF] = atom_idx`, `bas[ANG_OF] = l`, `bas[NPRIM_OF] = nprim`, `bas[NCTR_OF] = nctr`, `bas[PTR_EXP] = offset`, `bas[PTR_COEFF] = offset`

    **For each operator (ovlp, kin, nuc):**

    1. **Get libcint reference values:** Call `cintx_compat::raw::eval_raw()` with the appropriate `RawApiId` for `int1e_ovlp_sph`, `int1e_kin_sph`, `int1e_nuc_sph` and the H2O STO-3G inputs. Collect all shell-pair blocks into a reference matrix.

    2. **Get cintx values:** Build the equivalent cintx pipeline:
       - Construct `BasisSet` from the same atomic/shell data
       - Create `ExecutionPlan` via `query_workspace` + `ExecutionPlan::new`
       - Call `CubeClExecutor::new().execute(&plan, &mut io)`

    3. **Compare elementwise:**
       ```rust
       let atol = 1e-11_f64;  // TOL_1E_ATOL
       let rtol = 1e-9_f64;   // TOL_1E_RTOL
       let mut mismatch_count = 0;
       for (i, (&cintx_val, &libcint_val)) in cintx_output.iter().zip(libcint_output.iter()).enumerate() {
           let diff = (cintx_val - libcint_val).abs();
           let threshold = atol + rtol * libcint_val.abs();
           if diff > threshold {
               mismatch_count += 1;
               eprintln!("MISMATCH at index {i}: cintx={cintx_val:.15e}, libcint={libcint_val:.15e}, diff={diff:.3e}");
           }
       }
       assert_eq!(mismatch_count, 0, "Oracle parity failed with {mismatch_count} mismatches");
       ```

